DISTANBOL

Distanbol accepts two parameters as query parameters:
URL: This is the URL to the Stanbol output json.
Confidence(Optional): This is the minimum confidence threshold. The results will have a confidence of greater than this number. The default value is 0.7.
You can either:
  • Put your Stanbol json output URL as a query: 

    https://distanbol.acdh.oeaw.ac.at?URL={URL of your Stanbol output in json}&confidence={minimum confidence threshold}


  • Or use this form:
    URL: Confidence(Optional):

Examples with default Confidence(0.7):
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example.json
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example1.json
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example2.json
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example3.json
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example4.json
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example5.json

Examples with confidence as parameter(same example, different confidence):
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example5.json&confidence=0.5
  • https://distanbol.acdh.oeaw.ac.at?URL=https://distanbol.acdh.oeaw.ac.at/example/example5.json&confidence=0.7
